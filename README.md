# 📰 DAYNEW

**개인 맞춤 뉴스 서비스**

DAYNEW는 관심사 기반으로 뉴스를 수집하고 제공하는 웹 서비스입니다. 
다양한 뉴스 소스에서 최대 50개의 뉴스를 수집하여 사용자에게 맞춤형 뉴스 경험을 제공합니다.

## 🎯 프로젝트 개요

DAYNEW는 다음과 같은 문제를 해결합니다:
- **뉴스 정보 과부하**: 너무 많은 뉴스 중에서 관심 있는 내용만 선별
- **시간 절약**: 관심 있는 뉴스만 선별하여 효율적인 정보 수집
- **개인화**: 사용자의 관심사와 키워드에 맞는 뉴스만 제공
- **다양성**: 여러 뉴스 소스에서 다양한 관점의 뉴스 수집

## 🚀 주요 기능

### 📰 뉴스 수집
- **다중 API 연동**: NewsAPI + GNews API로 다양한 뉴스 소스 활용
- **최대 50개 뉴스**: 풍부한 뉴스 콘텐츠 제공
- **실시간 수집**: 최신 뉴스를 자동으로 가져옴
- **중복 제거**: 제목과 URL 기반으로 중복 뉴스 자동 제거

### 🔍 검색 및 필터링
- **카테고리 필터링**: IT/기술, 경제, 스포츠, 정치, 연예 등
- **키워드 검색**: 원하는 키워드로 정확한 뉴스 검색
- **독립적 검색**: 카테고리 무시하고 키워드만으로 검색 가능

### 📰 뉴스 표시
- **원문 표시**: 뉴스의 원본 내용을 그대로 제공
- **이미지 지원**: 뉴스 이미지와 함께 시각적으로 풍부한 표시
- **소스 정보**: 뉴스 출처와 발행 시간 표시

## 📦 설치 및 실행

### 1. 저장소 클론
```bash
git clone <repository-url>
cd vibecoding_1
```

### 2. Python 의존성 설치
```bash
pip install -r requirements.txt
```

### 3. API 키 설정
`env_example.txt` 파일을 참고하여 `.env` 파일을 생성하고 API 키를 설정하세요:

```bash
# .env 파일 생성
NEWS_API_KEY=your_news_api_key_here
```

#### API 키 발급 방법:
- **NewsAPI**: https://newsapi.org/ (무료)

### 4. 서버 실행
```bash
# 백엔드 서버 실행 (포트 5000)
python app.py

# 프론트엔드 서버 실행 (포트 8000) - 새 터미널에서
python -m http.server 8000
```

### 5. 웹페이지 접속
브라우저에서 `http://localhost:8000`으로 접속하세요!

## 🎯 사용법


### 기본 뉴스 보기
1. **관심사 설정**: 원하는 카테고리 선택 (IT/기술, 경제, 스포츠, 정치, 연예)
2. **키워드 입력**: 관심 있는 키워드 입력 (예: AI, 코인, 축구, 부동산)
3. **뉴스 업데이트**: "📰 뉴스 업데이트" 버튼 클릭
4. **뉴스 확인**: 최대 50개의 뉴스를 카드 형태로 확인

### 키워드 검색
1. **키워드 입력**: 검색하고 싶은 키워드 입력
2. **검색 실행**: "🔍 검색" 버튼 클릭
3. **결과 확인**: 해당 키워드와 관련된 뉴스만 필터링하여 표시
